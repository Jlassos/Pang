# Pang
## Description
Pong clone with network/online multiplayer.
C++ using [SFML](https://www.sfml-dev.org/) library.

# Required dlls to run

- sfml-audio-2.dll
- sfml-graphics-2.dll
- sfml-network-2.dll
- sfml-system-2.dll
- sfml-window-2.dll

- libgcc_s_seh-1.dll
- bstdc++-6.dll
- bwinpthread-1.dll
